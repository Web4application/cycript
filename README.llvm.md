# Web4application/cycript
A modernized fork of Cycript for the Web4application ecosystem.

## Features
- **ES6+ Support**: Native `async/await`, arrow functions, and more.
- **Modern Injection**: Powered by Frida-Gum for iOS 15+ and Android 12+ compatibility.
- **Rootless Ready**: Fully compatible with rootless jailbreaks.
- **Automated Builds**: Continuous Integration via GitHub Actions.

## Usage
`cycript -p [PID]`
